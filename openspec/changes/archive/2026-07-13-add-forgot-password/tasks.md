- [x] 1.1 New `api/src/services/email.rs`: `EmailSender` trait (`async fn send(&self, to: &str, subject: &str, html_body: &str) -> Result<(), EmailSendError>`), `EmailSendError` enum, `SharedEmailSender` type alias — mirrors `services/reverse_geocoder.rs`'s shape.
